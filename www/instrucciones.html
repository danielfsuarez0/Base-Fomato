Acerca de InterLab

InterLab es una aplicación diseñada para buscar y procesar datos de pruebas de desempeño realizadas en el laboratorio. Está estructurada en dos módulos principales:

  • Módulo IDEAM: Aquí se analizan las técnicas utilizadas en las pruebas interlaboratorio según los estándares del IDEAM.
  • Módulo ONAC: En este módulo se encuentran las técnicas de pruebas interlaboratorio bajo la normativa del ONAC.

Módulo de Indicadores

Dentro de cada módulo, se dispone de un área de indicadores donde se presentan gráficas de barras para analizar los resultados de las pruebas:

  • Pruebas Satisfactorias: Indican el porcentaje de pruebas que cumplen con los criterios establecidos.
  • Pruebas Insatisfactorias: Representan las pruebas que no alcanzan los estándares de calidad requeridos.
  • Pruebas Cuestionables: Son aquellas cuyos resultados generan dudas o necesitan revisión adicional.

InterLab proporciona herramientas intuitivas para interpretar y visualizar estos indicadores, permitiendo un análisis completo de los datos de desempeño del laboratorio.
